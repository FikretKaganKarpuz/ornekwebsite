# ornekwebsite Fikret Kağan Karpuz - Veri Bilimi ve Yapay Zeka PortfolyosuBu depo, Işık Üniversitesi Yönetim Bilişim Sistemleri öğrencisi Fikret Kağan Karpuz'un kişisel portfolyo web sitesinin kaynak kodlarını içermektedir. Proje, veri bilimi ve yapay zeka alanındaki çalışmaları sergilemek amacıyla, modern web tasarım prensipleri ve kullanıcı deneyimi odaklı bir yaklaşımla geliştirilmiştir.Tasarım dili olarak "Glassmorphism" (buzlu cam) estetiği benimsenmiş olup, site içerisinde dinamik imleç etkileşimleri ve özgün animasyonlar kullanılmıştır.Proje ÖzellikleriBu web sitesi aşağıdaki teknik ve tasarım özelliklerini barındırmaktadır:Modern Arayüz Tasarımı: Koyu mod tabanlı, gradyan geçişli arka plan üzerinde kurgulanmış yarı saydam kart yapıları.İmleç Etkileşimi (Cursor Glow): Kullanıcı faresi ile eş zamanlı hareket eden ve odaklanılan öğeler üzerinde görsel geri bildirim sağlayan dinamik aydınlatma efekti.Duyarlı Tasarım (Responsive): Mobil, tablet ve masaüstü cihazlarda tam uyumluluk sağlayan esnek ızgara (grid) sistemi ve mobil uyumlu navigasyon menüsü.Tek Dosya Mimarisi: Dağıtım ve kullanım kolaylığı sağlamak adına HTML, CSS ve JavaScript kodlarının optimize edilerek tek bir dosyada birleştirilmesi.İçerik Bölümleri:Giriş: Karşılama ekranı ve sosyal medya yönlendirmeleri.Hakkımda: Akademik geçmiş, biyografi ve teknik yetkinlik detayları (Python, SQL, Makine Öğrenmesi).Portfolyo: Hugging Face, Kaggle ve GitHub üzerindeki projelerin sergilendiği interaktif proje kartları.İletişim: Ziyaretçi etkileşimi için ön yüz tasarımı tamamlanmış iletişim formu.Kullanılan TeknolojilerProjenin geliştirilmesinde aşağıdaki teknolojiler ve kütüphaneler kullanılmıştır:HTML5: Semantik sayfa iskeleti.CSS3: Flexbox ve Grid yerleşim sistemleri, CSS değişkenleri, backdrop-filter efektleri ve anahtar kare (keyframe) animasyonları.JavaScript (Vanilla): DOM manipülasyonu, olay dinleyicileri (event listeners), mobil menü kontrolü ve imleç takip algoritmaları.Font Awesome: Kullanıcı arayüzü ikonları.Google Fonts: Tipografi için 'Poppins' yazı tipi ailesi.Kurulum ve ÇalıştırmaBu proje herhangi bir sunucu tarafı bağımlılığı veya derleme işlemi gerektirmez. Doğrudan tarayıcı üzerinde çalıştırılabilir.Projeyi yerel makinenize klonlayın:git clone [https://github.com/FikretKaganKarpuz/portfolyo.git](https://github.com/FikretKaganKarpuz/portfolyo.git)
